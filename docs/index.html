Easy & Fair Peer Judging for Hackathons

Podium Pro is an open-source tool for running a better hackathon peer-judging process. Created by Eddy Zhao (@eddyzow) to make organizing hackathons that much easier. Inspired by Hack Club's Podium tool.

How It Works

For Attendees

Designed to be simple, fair, and secure.

1. Join Securely

Enter the code provided by the event organizer, provide your information, and log in via magic link.

2. Add Your Project

Create your project, add documentation, and ship seamlessly. Only one team member needs to do this.

3. Vote Fairly

A simple swipe-left, swipe-right interface prevents ties and makes voting fast, fun, and unbiased.

For Organizers

Get your event up and running in minutes.

1. Generate Event Code

Sign in and generate an event code for your hackathon. Share this only with attendees.

2. Manage Results

Our voting algorithm automatically and fairly determines results in real-time. Freely view and moderate them.

3. Seamless Awards

When it's time for awards, display an automatically-generated slideshow to announce the winners.
